# Task 2.4: Registration Form with Validation

Create a React TypeScript registration form with:
- Email field (validate email format)
- Password field (minimum 8 characters)
- Confirm Password field (must match password)

## Requirements
- Show errors below each field
- Submit button enabled only when form is valid
- Validate on blur (onBlur)
- On successful submit, show alert('Success!')

## Expected file
Create `RegistrationForm.tsx` in this directory.
