In the original code, when adding an existing item to the cart:
```tsx

```

This directly mutates the `items` array element, and then tries to set the state to the same mutated array. React's

```tsx

```

quantity in the copy, and finally set the state to this new array. This creates a new reference, which React detects as 
a state change, triggering a re-render.
