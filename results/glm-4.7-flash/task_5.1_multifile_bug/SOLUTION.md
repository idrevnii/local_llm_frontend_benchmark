# Решение проблемы с сохранением корзины

## Проблема

После обновления страницы корзина очищалась, хотя в localStorage данные сохранялись.

## Причина бага

В файле `src/hooks/useLocalStorage.ts` хук использовал `JSON.parse()` для десериализации данных из localStorage. Когда данные содержали объекты `Date` (поля `addedAt` в `CartItem` и `updatedAt` в `Cart`), `JSON.parse()` конвертировал их в строки (ISO формат), а не восстанавливал обратно в объекты `Date`.

Когда код пытался использовать эти поля как даты, они были строками, что приводило к некорректной работе приложения.

## Решение

В функции `JSON.parse()` добавлен второй параметр - reviver функция, которая проверяет каждое значение. Если значение - это строка, похожая на дату (формат ISO), она конвертируется обратно в объект `Date`.
