# Компонент Countdown Timer

Создай React TypeScript компонент таймера обратного отсчёта.

## Требования

### Функциональность
1. Принимает prop `endTime: Date` — время окончания
2. Показывает оставшееся время в формате "2д 5ч 30м 15с"
3. Обновляется каждую секунду
4. Когда время истекло, показывает "Время вышло!"

### Архитектурные требования
5. Компонент должен быть **pure** (без side effects)
6. Компонент должен работать на **сервере (SSR)**
7. Должен учитывать **timezone пользователя**

### Технические требования
8. TypeScript с полной типизацией
9. Функциональный компонент
10. **Нельзя использовать внешние библиотеки** (moment, date-fns, etc.)

## Создай файл

`CountdownTimer.tsx`

## Props Interface

```typescript
interface CountdownTimerProps {
  endTime: Date;
  onComplete?: () => void;
  className?: string;
}
```

## Ожидаемый вывод

- До окончания: "2д 5ч 30м 15с"
- После окончания: "Время вышло!"
