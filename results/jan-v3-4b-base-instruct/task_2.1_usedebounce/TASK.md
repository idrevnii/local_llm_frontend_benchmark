# Task 2.1: useDebounce Hook

Implement a custom React hook `useDebounce<T>(value: T, delay: number): T` that returns a debounced value.

## Requirements
- Use useEffect and setTimeout
- Properly clean up the timer on unmount
- TypeScript generics for type safety
- Reset timer when value changes

## Expected file
Create `useDebounce.ts` in this directory.
