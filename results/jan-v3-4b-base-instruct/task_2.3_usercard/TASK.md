# Task 2.3: UserCard Component

Create a TypeScript UserCard component that:
- Accepts a user object: `{ name: string, email: string, avatar?: string, role: 'user' | 'admin' }`
- If avatar exists — show an image
- If avatar is missing — show initials (first letters of name parts)
- If role === 'admin' — show an "Admin" badge next to the name

## Expected file
Create `UserCard.tsx` in this directory.
