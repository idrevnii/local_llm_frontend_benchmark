# Task 1.1: Button Component

Create a React Button component in TypeScript with the following props:
- `text` (string) — button text
- `onClick` (function) — click handler  
- `disabled` (boolean, optional) — disabled state

Export the component as default.

## Requirements
- Use TypeScript with proper type definitions
- Component should compile without errors
- disabled prop should properly disable the button

## Expected file
Create `Button.tsx` in this directory.
