# Bug Report: Корзина не сохраняется между сессиями

## Описание проблемы
Пользователи жалуются, что после обновления страницы корзина очищается,
хотя в localStorage данные вроде бы сохраняются.

## Шаги воспроизведения
1. Добавить товар в корзину
2. Проверить localStorage — данные там есть
3. Обновить страницу
4. Корзина пустая, хотя в localStorage данные остались

## Структура проекта
```
src/
├── types/
│   └── cart.ts          # Типы данных
├── hooks/
│   └── useLocalStorage.ts   # Хук для localStorage
├── context/
│   └── CartContext.tsx      # Контекст корзины
└── components/
    └── CartIcon.tsx         # Иконка корзины
```
