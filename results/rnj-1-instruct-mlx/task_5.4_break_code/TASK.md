# Code Review: Найди баги через тесты

Разработчик уверяет, что функция `parseUserInput` полностью работает и покрывает все случаи.

**Твоя задача:** Докажи обратное! Напиши тесты, которые сломают этот код.

## Файл для тестирования

Смотри файл `src/parseUserInput.ts`

## Формат входных данных

Строка в формате:
```
"name:John;age:25;email:john@example.com;tags:developer,senior"
```

- Поля разделены точкой с запятой `;`
- Ключ и значение разделены двоеточием `:`
- Теги разделены запятой `,`

## Требования к тестам

1. Используй **Jest** синтаксис
2. Найди **минимум 8 случаев**, где функция работает неправильно
3. Каждый тест должен иметь **понятное название**
4. Объясни в комментариях, **почему это баг**
